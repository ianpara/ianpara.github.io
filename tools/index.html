Web Tools

Tools to open a list of URL, find words in URLS, or create custom open URL functions.

These tools require pop-up's to work. Click this button to test if you have pop-ups disabled.

Popup Test

In Chrome: If pop-ups are blocked you will see an icon in you address bar. Click that a select 'Always allow...'

In Firefox: If pop-ups are blocked you will see an alert telling you to enable pop-ups. In the top bar click 'Options' then 'Allow...'

In IE: If pop-ups are blocked you will see an alert telling you to enable pop-ups. In the bottom bar click 'Allow...'
